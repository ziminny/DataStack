## 🛠 Installation

### Swift Package Manager
```swift
.package(
  url: "https://github.com/ziminny/Datastack.git",
  from: "1.0.0"
)
```

### Cocoapods
```ruby
pod 'Datastack', :git => 'https://github.com/ziminny/Datastack.git'
```
